# Disclaimer

`regionhop` is a personal, self-hosted networking tool. Please read this before
using it.

## What regionhop is

- A command-line utility that opens an **SSH SOCKS5 tunnel to a virtual machine
  that you own and control**, and routes a browser or `yt-dlp` through it.
- Functionally equivalent to running your own VPN/proxy on your own server.
